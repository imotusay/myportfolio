# 安冨 秀和（Hidekazu Yasutomi）- ポートフォリオ

建設業界での徹底した工程管理と泥臭い問題解決力を強みに、IT業界へと転身しました。
現在はモダンな開発環境のもと、スピード感を持ったWEB系エンジニアとして、フロントエンドからバックエンド、業務効率化システムの構築まで一気通貫で対応しています。

また、AIを活用した動画編集ツール「Vrew」を熟知しており、迅速なテロップ生成やカット編集など、動画を用いたビジネスの訴求力向上にも貢献できます。

---

## 🛠 技術スタック

- **Frontend:** HTML5, CSS3, JavaScript (ES6+), レスポンシブWebデザイン
- **Backend / DB:** Webアプリケーション開発、リレーショナルデータベース設計（RDB）
- **Video Editing:** Vrew（AI動画編集ツールによる高速編集・テロップ生成）
- **Tools / Infrastructure:** Git, GitHub, ドメイン取得・DNS設定, サーバー運用保守（さくらのレンタルサーバ等）

## 🚀 開発実績

### 1. LOVOTカフェ 公式サイト
癒やしロボット「LOVOT」と触れ合えるカフェの公式Webサイトを構築しました。
- **URL:** [https://lovotcafe.com](https://lovotcafe.com)
- **担当範囲:** 要件定義、ドメイン取得、サーバー選定・構築、フロントエンド、バックエンド
- **詳細:** 予約導線やメニュー・グッズ紹介へのアクセス性を重視。デザインの再現性にこだわり、スマートフォン対応（レスポンシブ）を完璧に施しました。

### 2. カラオケ機器回収・修理管理システム
カラオケ事業から撤退した企業から引き取ったマイクやリモコン等の機器を、分解・リサイクル・修理するための社内業務システムです。
- **担当範囲:** 要件定義、DB設計、バックエンド、フロントエンド
- **詳細:** 手作業だった在庫・進捗管理をデジタル化。「修理再生用」と「素材リサイクル用」の仕分け・追跡を可能にし、機材の再利用率向上と工程ごとのトレーサビリティを確保。現場の作業効率化とサステナビリティ（資源循環）を実現しました。

### 3. 菌床栽培 収穫量管理システム「みらい絆農園」
菌床栽培で収穫された農産物の収穫量をリアルタイムで管理・集計するWebアプリケーションです。
- **URL:** [https://sakura.ne.jp](https://sakura.ne.jp)
- **担当範囲:** 要件定義、DB設計、バックエンド、フロントエンド構築、サーバー運用
- **詳細:** 日別・月別・棚別・人別の高度なデータ集計機能に加え、菌床棚配置図、収穫カレンダー等の機能を実装。アナログなデータ管理をデジタル化し、多角的な集計データをもとにした栽培計画の最適化と、農作業現場のDX（デジタルトランスフォーメーション）を推進しました。

## 📌 大切にしていること（信条）

1. **バグのない堅牢なコードと工程管理**
   建設業界で培った徹底した「安全・品質管理」の意識をコーディングにも適用し、堅牢なシステム構築を徹底します。
2. **期日遵守と迅速なコミュニケーション**
   プロジェクトの進捗管理（マイルストーン意識）を徹底し、ビジネスのスピード感を落とさない開発を行います。
3. **現場主義のDX推進**
   カラオケ機器管理や農園管理システムのように、実際の現場で働く人が迷わず使え、確実に労働生産性が向上するUI/UX・システム設計を追求します。
